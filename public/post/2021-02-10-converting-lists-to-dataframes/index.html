converting lists to dataframes

2021-02-10
R

Intro

Lists are tricky.

Create a list

my.l <- list(a = 1:2, b = 1:3, c = 1:4)
my.l
## $a
## [1] 1 2
## 
## $b
## [1] 1 2 3
## 
## $c
## [1] 1 2 3 4

Convert to Dataframe

# convert to df

df <- data.frame(name = names(my.l),
                 values = unlist(lapply(my.l, "[[", 1)))

df
##   name values
## a    a      1
## b    b      1
## c    c      1

Conclusion

That’s how you do it.
